01 — Profil

Men haqimda

Men Toshkentda joylashgan data analitikman — kompaniyaning CRM/ERP tizimlaridan olingan ma'lumotlarni tahlil qilib, savdo hisobotlari va boshqaruv dashboardlarini tayyorlayman.

Ish jarayonimda ko'proq Excel va Power BI bilan chuqur ishlayman — DAX formulalar, Power Query orqali ma'lumotlarni tozalash, va vizual hisobotlar qurish. Shu bilan birga Python va SQL orqali katta hajmdagi ma'lumotlarni qayta ishlashni ham o'rganmoqdaman.

Bilganlarimni o'zimda saqlab qo'ymayman — YouTube kanalimda o'zbek tilida Excel, Power BI va Python bo'yicha darslar yarataman, real hayotiy datasetlar asosida.
